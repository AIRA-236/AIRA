## Development

### Prerequisites
- Node.js >= 16
- Python >= 3.8
- CUDA >= 11.0 (for GPU support)

### Quick Start
```bash
# Clone repository
git clone https://github.com/AIRA-236/AIRA.git
cd AIRA

# Install dependencies
npm install

# Start development
npm run dev
```
